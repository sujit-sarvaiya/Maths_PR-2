## Project Overview

This project analyzes the probability of students passing a mathematics
exam using a generated dataset of 200 students. The dataset includes
information about study hours, attendance, participation in group
discussions, previous test scores, and final exam results.

The goal of the project is to understand how probability concepts can be
applied in real-world data analysis to estimate exam success.

------------------------------------------------------------------------

## Dataset Features

The dataset contains the following columns:

-   **study_hours** -- Number of hours a student studies per week\
-   **attendance** -- Student attendance percentage\
-   **group_discussion** -- Participation in group discussion (Yes/No)\
-   **previous_test_score** -- Marks obtained in previous mathematics
    test\
-   **final_exam_pass** -- Final exam result (Pass/Fail)

A dataset of **200 students** was generated using Python.

## Probability Concepts Used

This project demonstrates several important probability concepts:

### 1. Empirical Probability

Calculated using real dataset observations.

### 2. Theoretical Probability

Calculated using mathematical expectations.

### 3. Random Variable

Number of students passing the exam out of randomly selected students.

### 4. Probability Distribution

Used to analyze possible outcomes and their probabilities.

### 5. Mean and Variance

Used to measure expected value and variability.

## Data Visualization

Multiple charts were created to analyze the dataset:

-   Study Hours Distribution Histogram\
-   Previous Test Score Histogram\
-   Attendance vs Pass Bar Chart\

These visualizations help understand patterns between student behavior

------------------------------------------------------------------------

## Venn Diagram Analysis

A Venn diagram was used to analyze the relationship between:

-   Students studying more than 10 hours
-   Students with attendance above 80%

The overlapping section represents students who satisfy both conditions
and have higher chances of passing the exam.

------------------------------------------------------------------------

## Contingency Table Analysis

A contingency table was used to analyze the relationship between:

-   Group Discussion Participation
-   Final Exam Result

From this table, the following probabilities were calculated:

-   Joint Probability
-   Marginal Probability
-   Conditional Probability

------------------------------------------------------------------------

## Bayes Theorem Application

Bayes theorem was used to calculate the probability that a student
passed the exam given that the student had high attendance.

This helps update probability estimates when additional information is
available.

------------------------------------------------------------------------

## Key Findings

-   Students who study more hours tend to perform better.
-   High attendance significantly increases the chance of passing.
-   Participation in group discussions improves exam performance.
-   Previous test scores strongly influence final exam results.
